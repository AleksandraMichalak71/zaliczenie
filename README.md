# Projekt HTML – Strona z sekcją o lwie

## Opis projektu

Projekt przedstawia prostą stronę internetową stworzoną w języku HTML. W ramach rozwoju projektu dodano zdjęcie lwa zapisane w katalogu `img` oraz nową sekcję zawierającą informacje o lwie.

Celem projektu było zapoznanie się z podstawami pracy z repozytorium GitHub, zarządzaniem wersjami przy użyciu Git oraz współpracą z wykorzystaniem forków i branchy.

## Struktura projektu

```text
/
├── index.html
├── README.md
├── CONTRIBUTING.md
└── img/
    └── lew.jpg
    ...
```

## Wykorzystane technologie

* HTML5
* Git
* GitHub

## Wprowadzone zmiany

* dodanie obrazu lwa do katalogu `img`
* utworzenie nowej sekcji prezentującej lwa
* aktualizacja dokumentacji projektu w pliku README.md
* utworzenie pliku CONTRIBUTING.md opisującego zasady współpracy


Projekt wykonany w ramach zajęć z wykorzystania systemu kontroli wersji Git oraz platformy GitHub.
